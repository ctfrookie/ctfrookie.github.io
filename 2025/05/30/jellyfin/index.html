简介

Jellyfin 是一个基于社区开发的媒体服务器，支持多种平台（Windows、Linux、macOS、Docker 等）。它能够将本地媒体文件（如电影、音乐、照片）整理为美观的界面，并通过局域网或互联网进行流媒体播放。

配置 Jellyfin

初始设置向导

  1. 语言与地区设置
  2. 创建管理员账户
  3. 媒体路径设置（可跳过，后续手动添加）

    • 允许 Jellyfin 检测公网 IP（若需远程访问）
  5. 元数据语言与图像设置
  6. 完成设置

添加媒体库

  1. 登录 Jellyfin 后，点击顶部菜单栏的【媒体库】
  2. 点击【+ 添加媒体库】
  3. 选择媒体类型（电影、电视剧、音乐等）
  4. 设置媒体存放路径（建议使用本地磁盘路径，如 D:\Movies）
  5. 设置元数据匹配器（推荐使用 TheMovieDB 和 TheTVDB）
  6. 确认后 Jellyfin 将开始扫描媒体文件

用户管理

  1. 点击顶部菜单栏的【用户】
  2. 创建新用户或编辑已有用户权限
  3. 可以为不同用户分配不同的媒体库访问权限
  4. 支持家长控制、设备限制等功能

远程访问设置

  1. 进入后台设置 → 网络
  2. 勾选“启用远程访问”
  3. 如果处于家庭网络中，确保路由器已开启端口转发（默认 Jellyfin 端口：8096）
  4. 可以绑定自定义域名或使用 Jellyfin 提供的 .local.jellyfin.org 子域名

提示：可以使用反向代理（如 Nginx 或 Cloudflare Tunnel）来增强安全性与访问体验。

更新 Jellyfin

要升级到新版本：

  1. 停止当前运行的 Jellyfin 实例
  2. 备份配置文件（通常位于 %APPDATA%\Jellyfin\Server）
  3. 下载最新版本并替换旧文件
  4. 重新启动 jellyfin.exe

常见问题

Q: 启动时报错 Could not load file or assembly 'System.Runtime...

A: 请确认已安装 .NET Runtime 6.0 或更高版本。

Q: 无法访问 Jellyfin 网页？

A: 检查防火墙是否放行了 8096 端口，或尝试更换端口。

Q: 如何备份 Jellyfin 数据？

A: 备份 %APPDATA%\Jellyfin\Server 目录即可保存所有配置、用户信息和数据库。
